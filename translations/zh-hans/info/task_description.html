在本系列任务中，您将制作一个初级计算器。

作为输入，您需要按下一连串的键，而作为该功能的结果，您应显示按下最后一个键时屏幕上将显示的内容。注意，这并不总是表达式的结果。实际上，使用实物计算器或应用程序确实有助于你捕捉边缘情况。

在第七个任务中，内存应执行以下操作：大写 "A"（加，+）、大写 "S"（减，-）、大写 "G"（取值）和大写 "E"（擦除内存）。

预期行为

  • 应删除开头的零，将仅有零的数字转换为单个零；
  • 数字间的 +- 符号应取最后一个；
  • "=="表示重复上次操作；
  • "+="或"-="--在组合前对数字（或运算结果）进行加减运算（将数字加倍或减去本身）；
  • 计算器忽略在第 5 位之后输入的数字；
  • 数字 < 0 的"-"不占位数；
  • 如果 abs(结果)大于 99999，结果显示为 "错误"；
  • 对于浮点数，如果 abs(结果) 的整数部分大于 9999，则结果显示为 "错误"；
  • 对于 float，如果数字的总长度超过 5 位数，则应四舍五入到 5 位数（1.23456 -> 1.235）；
  • 对于浮点数，应去掉开头和尾部的零（如果可能，去掉". "之前的零）： 0.1200 -> .12 , 123.00 -> 123..应在四舍五入后进行： 1.000123 -> 1. .只有在输入数字结束后（按下非数字字符），才能删除尾部的零；
  • 当 B 字符后没有符号且新数字被按下时 - 与前一个数字连接：12+B3 -> 123 ；
  • A"/"S "按前一个数字增加/减少内存，并清除输出；
  • G "在操作后插入内存值，否则清洁输出并插入内存值。

输入：字符串。

输出： 字符串字符串。
